Qui suis-je?

Bonjour je suis LAURENT Silvére, apprenti développeur web


En projet dans une reconversion informatique, plus précisément dans le développement web depuis fin d' année 2024, j'ai commencé à m’ auto former via une application nommé « MIMO », j’ai aussi fait un stage en début 2025 chez un développeur web.

Pendant ce moment passé avec Mr STARCK Emmanuel, que je remercie fortement pour m’avoir accueilli, j’ai appris à utiliser des outils comme « BASH », « GIT », « GITHUB », « HTML », « CSS », « VS CODE », ainsi que des astuces pour développer aisément.

Suite à ce stage je continue à me former en autodidacte, en suivant des cours sur internet, je fais aussi mes propres recherches pour m’améliorer en lisant ou regardant des tutoriels sur différents outils qui m'ont été présentés afin de pouvoir de jour en jour, les utiliser de manière plus fluide.

L'avancement de mes connaissances se verront au fur et à mesure de l'évolution de mon portfolio, car je le mettrai à jour régulièrement maintenant que j' ai appris à créer un "commmit" sur github et à l' héberger sur un serveur.

Je vous souhaite une bonne visite sur mon portfolio et je vous laisse si vous le désirez me contacter via le formulaire de contact en bas de page.
